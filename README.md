# loan-predection
# 📊 Loan Prediction Project

## 📌 Overview

This project focuses on building a machine learning model to predict whether a loan application will be approved or not based on applicant details.
It involves data cleaning, exploratory data analysis (EDA), feature engineering, and model building using Python.

---

## 🚀 Objectives

* Analyze loan applicant data
* Identify key factors affecting loan approval
* Build a predictive model
* Evaluate model performance

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 📂 Dataset

The dataset includes information such as:

* Applicant Income
* Co-applicant Income
* Loan Amount
* Loan Term
* Credit History
* Property Area
* Loan Status (Target Variable)
